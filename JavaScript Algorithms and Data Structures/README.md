# JavaScript Algorithms and Data Structures

This folder contains my solutions and projects for the "JavaScript Algorithms and Data Structures" course from FreeCodeCamp. The course covers essential algorithms and data structures concepts using JavaScript.

## Course Curriculum

The course consists of several modules covering different topics. Here is an overview of the curriculum:

1. Basic JavaScript
2. ES6
3. Regular Expressions
4. Debugging
5. Basic Data Structures
6. Basic Algorithm Scripting
7. Object Oriented Programming
8. Functional Programming
9. Intermediate Algorithm Scripting
10. JavaScript Algorithms and Data Structures Projects
